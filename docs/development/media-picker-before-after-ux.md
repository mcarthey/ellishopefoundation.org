## Workflow Comparison

### Scenario: Upload a new event photo and use it

#### BEFORE (Old Experience):
```
1. Open Event Edit form
2. Scroll to Featured Image
3. Click "Browse Media Library"
4. Modal opens
5. See "Upload to Media Library" link
6. Click link ? NEW TAB OPENS
7. Navigate to Upload page
8. Fill upload form
9. Select file
10. Enter alt text
11. Select category
12. Click Upload
13. Wait for page reload
14. See success message
15. Close tab
16. Back to Event Edit (lost context)
17. Scroll to Featured Image again
18. Click "Browse Media Library" AGAIN
19. Modal opens again
20. Search for newly uploaded image
21. Find it in grid
22. Click to select
23. Modal closes
24. Save event

Total: 24 steps, 2 new tabs, 2 modal opens, context lost
```

#### AFTER (New Experience):
```
1. Open Event Edit form
2. Scroll to Featured Image
3. Click "Browse Media Library"
4. Modal opens with Browse tab active
5. Click "Upload New" tab
6. Select file ? Preview appears
7. Enter alt text
8. Click "Upload to Media Library"
9. Success message ? Auto-switches to Browse tab
10. See newly uploaded image in grid
11. Click to select
12. Modal closes
13. Save event

Total: 13 steps, 0 new tabs, 1 modal, context preserved
```

**Time Saved**: ~50%  
**Confusion Eliminated**: 100%  
**Tab Opens**: 0 (was 2)
