### Step 1: Log In
1. Go to your website admin panel

Click "Edit" next to the page you want to update:

## ?? **Common Questions**

### Q: I can't find the field I want to edit
**A:** Each page has specific editable areas. If you need to edit something not listed, ask your web developer to add it to the template.

### Q: My image looks stretched or weird
**A:** Use the recommended size shown in the description. For example, "Hero Banner (1800x600px)" should be exactly that size.

### Q: I made a mistake! How do I undo?
**A:** Just edit the field again and save the correct content. The old content is replaced.

### Q: Where are my changes?
**A:** Changes appear on the live website immediately after you click "Save" or "Update". Refresh your browser to see them.

### Q: Can I edit the same thing someone else is editing?
**A:** Yes, but the last person to save wins. Coordinate with your team to avoid conflicts.

### Q: What's the difference between "Text" and "Rich Text"?
**A:**
- **Text** = Plain text only, no formatting
- **Rich Text** = Can use bold, italic, links, etc.

## ?? **Best Practices**

### Writing Effective Content

**Headlines:**
- Keep short (5-10 words)
- Be specific
- Use action words
- Example: "Transform Your Health Journey"

**Descriptions:**
- Start with most important info
- Use bullet points for lists
- Keep paragraphs short (2-3 sentences)
- Example: "Our programs help you build strength, improve endurance, and gain confidence through personalized training and community support."

**Images:**
- Use high-quality photos
- Ensure proper dimensions
- Choose relevant images
- Consider mobile viewers

### Content Checklist

Before saving, ask yourself:
- [ ] Is the message clear?
- [ ] Does it represent our organization well?
- [ ] Is there any typo or grammar error?
- [ ] Does the image support the text?
- [ ] Will it make sense to visitors?
